### Interaction Notes

- **Role Toggle**: Switches between Sarah (Frontline) and VP Sales (Senior)
  - When Frontline: [+ New Objective] button is hidden
  - When Senior: All features available
- **Ideas List**: Click to open Idea Detail Screen
- **Objectives List**: Click to open Objective Detail Screen
- **Surfaced Alerts**: Click to open the related Idea Detail Screen
  - Newest at top, oldest at bottom
  - No dismiss functionality (scroll to see history)

| Section | Description |
|---------|-------------|
| **Summary** | Freeform description of the idea (always present) |
| **Challenge** | The problem or situation being addressed |
| **Approach** | The guiding policy to address the Challenge |
| **Coherent Actions** | Specific steps that implement the Approach |
| **Linked Objective** | Connection to organizational Objective(s) |

### Interaction Notes

- **Canvas (70%)**: Each section is independently editable
  - [âœ] Edit button opens inline editor for that section
  - User edits are visible to the agent via AG-UI
  - Agent can propose changes that appear as suggestions with [Accept] / [Edit] buttons
- **Chat (30%)**: Ongoing conversation with IdeaAssistantAgent
  - Agent observes Canvas changes in real-time
  - Agent can offer suggestions, ask clarifying questions
  - User can ask for help ("Can you suggest an Approach?")
- **Status Badge**: Shows Draft / Active / Archived
- **AG-UI Protocol**: Bidirectional sync between Canvas and Chat
  - User edits Canvas â†’ Agent sees and can respond
  - Agent suggests edit â†’ Appears in Canvas as pending suggestion
